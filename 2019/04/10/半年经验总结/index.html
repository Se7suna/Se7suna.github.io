半年经验总结

发表于 2019-04-10 | 更新于 2019-05-08 | 分类于 实战经验

半年经验总结

Vue 对象后添加属性

使用$set(obj, key, value)添加绑定效果

key 的值为string

Vue 数据改变后不渲染

注意改变数据需要使用 vue 提供的变异方法, 或者 Vue.set

node-sass 安装失败

不生成 map 文件

config — index.js productionSourceMap: false

font-family

单行省略号

多行省略号

保留两位小数

num.toFixed(2)

Vue 里使用箭头函数

需要注意this的指向问题

chormeADB 调试

chrome://inspect

日期转时间戳

注意 main.js 创建实例的时间

在实例上挂载方法时, 需要在实例创建之前挂载

金额单位为分

需要 n * 1000 / 10 来保证精度

Vue 项目静态资源

放在 static 文件夹中

动态图片路径 ./static/img/xxx

打包后背景图路径错误

在 build => utils.js => return ExtractTextPlugin.extract({}) 内添加 publicPath: '../../'

变量名

命名描述其目的和作用而不是描述它的值

字体渐变色

Router History 模式页面空白

router.js 配置服务器 base: /路径

img 正方形自适应

npm 设置淘宝源

npm config set registry https://registry.npm.taobao.org

通过component元素 动态绑定自定义组件

Vue 绑定事件默认传入实参 event, 也可以自己传入 $event

.passive 修饰符可提高移动端滚动性能

原理: 在添加事件的时候提前告诉浏览器: 不需要阻止默认行为. 减少了浏览器等待运行时间, 从而能更快的响应事件

ul, ol, table, select 内部元素有限制

为了防止自定义组件被作为无效内容提升到外部, 可以在有效元素上使用 :is 来绑定自定义组件

在字符串和单文件中该限制不存在!!

在自定义组件上监听原生事件

使用 .native 修饰符

自定义组件的 v-model

在组件的 model: { prop: 'value', evnet: 'change'} 配置 v-model 绑定的值和监听的事件

注意: model 里的 prop 仍然需要在 props 里声明

.sync 修饰符

  1. 子组件$emit更新
  1. 父组件传递 props(title) 并快捷监听$emit

slot 插槽

使用自定义组件时, 所有未在 v-slot 具名插槽中的内容都会放到 自定义组件的 slot 标签中

未命名的 slot 标签的 name 属性 默认值为 default

  1. slot标签 使用 name属性 来命名

  2. 在 slot标签 中的内容就是该插槽的默认值

  3. 使用 v-slot: xxx 来对应插槽

  4. 将自定义组件的数据绑定在 slot标签 上, 父组件 v-slot 属性用一个变量名接收 插槽 props

  5. 动态插槽名使用 []

  6. 插槽缩写 #

依赖注入

  1. provide 选项允许我们指定我们想要提供给后代组件的数据/方法

  2. 使用 inject 选项来接收指定的我们想要添加在这个实例上的属性

注入的 数据/方法 是非响应式的

强制更新

$forceuUpdate

它仅仅影响实例本身和插入插槽内容的子组件，而不是所有子组件。

静态内容缓存

  1. 在根元素上添加 v-once 属性以确保这些内容只计算一次然后缓存起来

  2. 使用 v-pre 跳过大量没有指令的节点会加快编译。

vm.\$nextTick()

路由映射相同组件会复用

如果需要重新触发组件的钩子函数 可以绑定唯一 :key 使组件重新加载

路由视图命名

用于同一个页面有多个 router-view 去展示对应的 component

name 默认值 default

component 写成一个对象 component: { name: componentName }

vm.\$on(event, callback)

$on 可以监听所有事件, 包括 $emit 触发的自定义事件

检测数据类型

Object.prototype.toString.call(X).slice(8, -1) => "String"

设置根元素继承特性

  1. 组件内inhertAttrs: false

    这个选项不影响 class 和 style 绑定。

  2. 其他 html元素 通过 v-bind="$attrs" 来接收根元素的继承

项目打包优化

  1. 后端开启 gzip
  2. npm run build --report 分析包大小
  3. 项目开启 productionGzip
  • npm install --save-dev compression-webpack-plugin
  • package.json 配置 "compression-webpack-plugin": "^1.1.12"
  • prod.conf.js 修改 asset => filename
  1. 配置 externals 使用 CDN 加载插件/库
  • 在 base.conf.js 中配置 externals
  • 引入由import 改为 require('XXX')
  • index 页面 script标签 引入 CDN
  1. 路由组件懒加载

引入组件由 import 改为 require('url')

  1. 图片懒加载 / 使用 ps 压缩图片

Linux 下 install 报错 EACCES: permission denied

npm install --unsafe-perm=true --allow-root

JS 压缩图片

router-link标签 属性（props）

  1. append 默认 false 设置后相当于在当前路径之后添加新路径
  2. tag 设置 router-link 渲染后的标签
  3. active-class 设置链接激活后的样式名称
  4. event 设置触发跳转的事件

scrollBehavior

自定义路由切换时页面如何滚动

浏览器需要支持 history.pushState

router.addRoutes(routes: Array\<RouteConfig>)

动态添加更多的路由规则

router.onReady(callback, [errorCallback])

在路由完成初始导航时调用

router.onError(callback)

注册一个回调，该回调会在路由导航过程中出错时被调用
