发表于 2018-10-02 | 更新于 2019-05-08 | 分类于 SCSS

SASS

安装

npm install sass-loader node-sass --save-dev

变量

  • 所有变量以$开头
  • 如果变量需要镶嵌在字符串之中，就必须需要写在#{}之中

计算功能

  • sass 中可以使用加减乘除百分比计算值
  • 只有第一个数值需要带单位, 后面的都不需要单位

嵌套

  • 可以选择器嵌套
  • 也可以属性嵌套

继承

class2要继承class1，就要使用@extend命令

Mixin

使用@mixin命令，定义一个代码块

使用@include命令，调用这个mixin

可以指定参数和缺省值

浏览器兼容混合
