jQuery笔记

发表于 2018-09-10 | 更新于 2018-09-16 | 分类于 JavaScript

jQuery

基础

了解jQuery

  1. 封装简化DOM操作
  2. 它有强大选择器 : 方便查找 DOM 元素
  3. 它能隐式便利 : 一次操作所有查找到的元素
  4. 它能链式调用 : jQ方法都会返回jQ对象
  5. 它可以读写合一 : 不传值则读取 传值则写入
  6. 事件处理 , DOM操作 , CSS操作 , 动画效果 , 浏览器兼容

引入jQuery

  1. 开发测试的时候使用未压缩的版本方便调试 (服务器本地引入)
  2. 正式上线的时候使用压缩版本加载速度快 (CDN远程引入)

jQuery两把利器

  • jQuery(核心)函数

    $()使用()调用时是当作函数使用

    1. 参数为函数 : 当DOM加载完成后,执行此回调函数
    2. 参数为选择器字符串 : 查找所有匹配的标签,并将他们封装为jQuery对象
    3. 参数为DOM对象 : 将DOM对象封装成为 jQuery对象
    4. 参数为HTML标签字符串 : 创建标签对象并封装成jQuery对象

    $.xxx() 使用 . 来使用方法时当作对象来使用

  • jQuery(核心)对象

    1. 执行jQuery核心函数返回的对象(函数名前加 $)

    2. jQuery对象内部包含的是 DOM 元素对象的伪数组(即使只有一个元素)

    3. jQuery对象拥有很多有用的属性和方法,让程序员能方便的操作 DOM

      语法: $obj.xxx()或者$("选择器").xxx()

基本用法

  1. .length 获取jQuery对象里面包含的 DOM 元素的个数
  2. [index] 返回index下标对应的 DOM 元素
  3. .each() 遍历jQuery对象里面的 DOM 元素
  4. .index() 得到 DOM 元素在其兄弟元素中的下标

选择器

jQuery选择器

jQ选择器基于css选择器, 并对其进行了扩展

作用: 根据选择器规则在整个文档中查找所有匹配的标签, 并封装成jQuery对象返回

选择器分类

  • 基本选择器: 和css选择器规则相同

    #id / element / .class / * / s1,s2(并集) / s1s2(交集)

  • 层次选择器: 查找子元素 / 后代元素 / 兄弟元素

    1. s1 s2 匹配s1后代所有的s2元素
    2. parent>child 匹配parent的子元素child
    3. prev+next 匹配prev后紧接的兄弟元素next
    4. prev~siblings 匹配prev之后的所有兄弟元素siblings

  • 过滤选择器: 在原有选择器匹配的元素中进行进一步的筛选

    多个过滤选择器是依次执行的, 需要注意index值的变化

    • 基本

      1. :first 选择第一个元素
      2. :last 选择最后一个元素
      3. :not(selector) 不匹配selector的元素
      4. :even 索引值为偶数的元素
      5. :odd 索引值为奇数的元素
      6. :eq(index) 指定索引值的元素
      7. :gt(index) 大于指定索引值的元素
      8. :lt(index) 小于指定索引值的元素
      9. :animated 正在执行动画效果的元素
      10. :focus 当前获取焦点的元素

    • 内容

      1. :contains(text) 匹配包含给定文本的元素
      2. :has(selector) 配含有选择器所匹配的元素的元素

    • 可见

      1. :hidden 匹配所有不可见元素
      2. :visible 匹配所有的可见元素

    • 属性

      1. : [attribute=value] 匹配给定的属性是某个特定值的元素
      2. [attribute!=value] 匹配所有不含有指定的属性

    • 子元素

      1. :nth-child(n) 匹配其父元素下的第N个子或奇偶元素

    • 表单

      1. :checkbox 匹配所有复选框
      2. :button 匹配所有按钮

工具方法

将 $ 当作对象来使用

  1. $.each(function("属性名/索引","属性值"){}): 遍历数组或对象中的数据
  2. $.trim(str): 去除字符串两边的空格
  3. $.type(obj): 得到数据的类型
  4. $.isArray(obj): 判断是否是数组
  5. $.isFunction(obj): 判断是否是函数
  6. $.parseJSON(json): 解析json字符串转换为js对象/数组

属性操作

  • 读取/设置任意属性

    1. attr("属性名","属性值") 单个属性直接传入”属性名”,”属性值”

      多个属性封装为对象传入 “属性名”:”属性值”

    2. removeAttr("属性名")

    3. prop("属性名",true/false) 操作属性值为布尔值的属性

  • 操作class类名

    1. addClass("类名") 为元素添加CSS类名，多个请用空格分开
    2. removeClass("类名") 删除元素的CSS类名，多个请用空格分开

  • 操作HTML

    1. html() 获取元素的内容,传参则设定HTML的内容
    2. val() 获得匹配元素的当前值,传参则设定值

CSS

  • .css()

    1. 传入”属性名” 则读取该属性
    2. 传入”属性名”,”值” 设置该属性
    3. 设置多个属性 封装为一个对象传入

  • 位置属性

    1. offset() 获取匹配元素相对页面左上角的偏移,只对可见元素有效

      返回的对象包含两个整型属性：top 和 left

      写入参数传入对象 {top:123,left:456}

    2. position() 获取匹配元素相对父元素的偏移

    3. scrollTop([val]) 获取匹配元素相对滚动条顶部的偏移

    4. scrollLeft([val]) 获取匹配元素相对滚动条左侧的偏移

  • 尺寸

    1. height() 内容高度
    2. width() 内容宽度
    3. innerHeight() height + padding
    4. innerWidth() width + padding
    5. outerHeight(true) height + padding + border + margin
    6. outerWidth(false) width + padding + border

对象筛选

过滤

在已匹配的元素对象中筛选出一个或多个组成新的jQuery对象返回

  1. eq(index|-index) 获取第N个jQuery对象，参数为负数时为反向选取
  2. first() last() 获取第一个或最后一个元素
  3. filter(selector) 筛选出当前元素中与选择器匹配的元素集合
  4. not(selector) 删除与选择器匹配的元素
  5. has(selector) 包含特定后代的元素，去掉那些不含有指定后代的元素

查找

通过已匹配的对象找到一个或多个新的元素对象封装为新的jQuery对象

  1. children(selector) 子标签中匹配
  2. find(selector) 后代标签中匹配
  3. parent(selector) 父标签中匹配
  4. prevAll(selector) 前面所有的兄弟标签中匹配
  5. nextAll(selector) 后面所有的兄弟标签中匹配
  6. siblings(selector) 前后所有的兄弟标签中匹配

文档处理

  • 添加

    1. append(content) / appendTo()

      向当前匹配的所有元素内部的最后插入指定内容

    2. prepend(content) / prependTo()

      向当前匹配的所有元素内部的最前面插入指定内容

    3. before(content)

      将指定内容插入到当前所有匹配元素的前面

    4. after(content)

      将指定内容插入到当前所有匹配元素的后面

  • 替换

    1. replaceWith(content)

      用指定内容替换所有匹配的标签删除节点

  • 删除

    1. empty()

      删除所有匹配元素的子元素

    2. remove()

      删除所有匹配的元素

事件处理

  • 事件绑定

    1. eventName(function(){})

      绑定对应事件名的监听, 例如：$(‘#div’).click(function(){})

    2. on(eventName, funcion(){})

      通用的绑定事件监听, 例如：$(‘#div’).on(‘click’,”a”, function(){})

    3. 常用事件

      click / mouseenter / mouseleave / focus / blur / hover

    4. 区别

      • eventName: 编码时绑定元素必须存在,不能给后添加的元素绑定该事件
      • on: 可以利用事件委派,将事件绑定在父元素上,事件可以冒泡到目标元素上

  • 事件解绑

    off(eventName)

  • 事件的坐标

    1. event.clientX,event.clientY 相对于浏览器窗口的左上角
    2. event.pageX,event.pageY 相对于当前页面的左上角
    3. event.offsetX, event.offsetY 相对于事件元素左上角

  • 事件处理

    1. event.stopPropagation() 停止事件冒泡
    2. event.preventDefault() 阻止事件默认行为

事件委托

  • 事件委托(委派/代理)

    1. 将多个子元素的事件监听委托给父辈元素处理
    2. 事件监听绑定在父辈元素上
    3. 当操作子元素时, 事件会冒泡到父辈元素
    4. 父辈元素不会直接处理事件, 而是根据event.target得到发生事件的子元素, 通过这个子元素调用事件回调函数

  • 事件委托的两方

    1. 委托方: 子元素
    2. 被委托方: 父元素

  • 使用事件委托的好处

    1. 添加新的子元素, 自动有事件响应处理
    2. 减少事件监听的数量: n==>1

  • jQuery的事件委托API

    1. 设置事件委托: $(parentSelector).delegate(childrenSelector, eventName, callback)
    2. 移除事件委托: $(parentSelector).undelegate(eventName)

动画效果

  • 淡入淡出: 不断改变元素的透明度opacity来实现的

    1. fadeIn("speed"): 带动画的显示
    2. fadeOut("normal"): 带动画隐藏
    3. fadeToggle(): 带动画切换显示/隐藏

  • 滑动动画: 不断改变元素的高度width来实现的

    1. slideDown("speed"): 带动画的展开
    2. slideUp("slow"): 带动画的收缩
    3. slideToggle(): 带动画的切换展开/收缩

  • 显示隐藏，默认没有动画

    通过同时改变宽 高 透明度来实现

    1. show("speed"): 不带动画的显示
    2. hide("fast"): 不带动画的隐藏
    3. toggle(): 不带动画的切换显示/隐藏

  • jQuery动画本质 : 在指定时间内不断改变元素样式值来实现的

    1. animate(params,[speed]): 自定义动画效果的动画

      params: 包含一个或多个需要操作属性和最终值的对象

    2. 移动指定距离 可以用{left : '+=100'} , {top : '-=50'}

    3. stop(): 停止动画

多库共存和插件

多库共存

  • 问题: 如果有2个库都有$, 就存在冲突
  • 解决: jQuery库可以释放$的使用权, 让另一个库可以正常使用, 此时jQuery库只能使用jQuery了

  • API: jQuery.noConflict()

  • window.onload与$(document).ready()的区别:

    • window.onload

      1. 包括页面的图片加载完后才会回调(晚)
      2. 只能有一个监听回调

    • $(document).ready()

      1. 等同于: $(function(){})
      2. 页面加载完就回调(早)
      3. 可以有多个监听回调

插件

  • 扩展jQuery的工具方法

    $.extend(object)

  • 扩展jQuery对象的方法

    $.fn.extend(object)

  • 将需要扩展的一个或多个方法封装成对象传入

    方法名: function(){}
